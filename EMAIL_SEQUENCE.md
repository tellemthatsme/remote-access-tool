## Email 2: Quick Tips (Day 3)
**Subject:** 3 tips to get the most out of RemotePC

```
Hey!

Hope you're enjoying RemotePC!

Here are a few tips:

1. Bookmark your URL
   Save your tunnel URL somewhere safe - you'll use it often!

2. Check the alerts
   If CPU or RAM goes above 80%, you'll see a warning. 
   That's your cue to kill some processes. ☠️

3. Use the kill buttons wisely
   - "Kill All Node" = stops Node, npm, npx, yarn
   - "Kill Docker" = stops Docker Desktop
   
   Great for freeing up memory when things get stuck!

Let me know if you have questions!

Cheers,
[Your Name]
```
