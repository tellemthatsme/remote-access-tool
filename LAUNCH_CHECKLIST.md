# Launch Checklist

## Pre-Launch (Day Before)
- [ ] Update landing.html with YOUR BuyMeACoffee link
- [ ] Test dashboard works locally (node dashboard.cjs)
- [ ] Test Cloudflare Tunnel (START_TUNNEL.bat)
- [ ] Take screenshots of dashboard
- [ ] Prepare all social media posts (copy-paste ready)

## Launch Day
- [ ] Create BuyMeACoffee page (5 min)
- [ ] Update landing.html link
- [ ] Post on Twitter/X
- [ ] Post on Reddit (r/programming)
- [ ] Post on Reddit (r/selfhosted)
- [ ] Post on LinkedIn
- [ ] Submit to Hacker News (if ready)
- [ ] Tell 5 friends personally

## Post-Launch (Week 1)
- [ ] Check for new subscribers daily
- [ ] Reply to all comments/messages
- [ ] Post follow-up on Twitter
- [ ] Thank early supporters personally
- [ ] Track analytics (views, clicks)

## Week 2-4
- [ ] Post on more subreddits
- [ ] Respond to feedback
- [ ] Make improvements based on feedback
- [ ] Consider YouTube tutorial
- [ ] Start building email list

## Content Calendar

### Week 1
- Day 1: Main launch posts
- Day 2: Answer questions, engage
- Day 3: Share "why I built this" story
- Day 4: Post feature highlight
- Day 5: Behind the scenes
- Day 6: Share feedback received
- Day 7: Weekly update

- Share improvements made
- Post comparison (vs TeamViewer)
